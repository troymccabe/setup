# setup

Personal Mac bootstrap for a fresh macOS Tahoe install.

## Quick start

```sh
curl -fsSL https://raw.githubusercontent.com/troymccabe/setup/main/mac/dev/bootstrap | sh
```

### Dry run (preview changes without applying them)

```sh
curl -fsSL https://raw.githubusercontent.com/troymccabe/setup/main/mac/dev/bootstrap | sh -s -- --dry-run
```

### Manual run (after cloning)

```sh
git clone https://github.com/troymccabe/setup.git ~/Developer/setup
cd ~/Developer/setup/mac/dev
chmod +x setup && ./setup
```

## Maintaining

### After changing `Brewfile` or `setup`, regenerate `SHA256SUMS`

```sh
shasum -a 256 Brewfile setup avatar.png wallpaper.png > SHA256SUMS
git add Brewfile setup.sh SHA256SUMS Brewfile.lock.json
git commit -m "chore: update setup"
```

### Upgrade Homebrew packages and commit the new lockfile

```sh
brew bundle --file=Brewfile          # updates Brewfile.lock.json
git add Brewfile.lock.json
git commit -m "chore: bump Brewfile lockfile"
```

### Bump a pinned runtime version

Edit the `MISE_VERSIONS` map near the top of ¬ß7 in `setup.sh`:

```bash
declare -A MISE_VERSIONS=(
  [dotnet]="9"
  [go]="1.23"
  [node]="22"
  [python]="3.13"
  [ruby]="3.3"
  [rust]="stable"
)
```

Then regenerate `SHA256SUMS` and commit.

## Shell config layout

`~/.zshrc` sources all `~/.config/zsh/*.conf` fragments automatically.

| File | Purpose |
|------|---------|
| `colors.conf` | `CLICOLOR=1` |
| `mise.conf` | `mise activate zsh` |
| `starship.conf` | `starship init zsh` |
| `dotnet.conf` | `DOTNET_ROOT`, `DOTNET_VERSION`, `MSBuildSDKsPath` (computed at startup) |
| `go.conf` | `GOROOT`, `PATH` addition (computed at startup) |

All paths in `dotnet.conf` and `go.conf` are computed via `mise which` at shell-startup time, so they automatically reflect whichever version `mise` has active ‚Äî no manual update needed when you switch runtime versions.

## Re-running

The script is fully idempotent ‚Äî safe to run again at any time:

- Homebrew packages are skipped if `brew bundle check` passes
- `gh auth login` is skipped if already authenticated
- mise installs are skipped or upgraded in-place
- Global npm/gem/cargo packages are skipped if already present
- Xcode download is skipped if the latest version is already installed
- `defaults write` calls are always safe to re-run (last-write wins)
- All `~/.zshrc` and `~/.config/zsh/*.conf` writes are idempotent
